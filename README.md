📅 Day 7 – Week 1 Reflection

Cyber Universe Task-Based Cybersecurity Internship

1️⃣ Which task helped you learn the most and why?

Day 6 – Phishing Email Analysis helped me learn the most. It made me think like an attacker and understand how social engineering works. Identifying red flags like urgency, suspicious links, and fake authority made me realize that cybersecurity is not just technical — it also involves human psychology. Now I analyze emails more carefully instead of trusting them immediately.

2️⃣ One concept that became clearer for you this week.

Linux file permissions (Day 2) became much clearer. Earlier, I only knew basic commands, but after using ls -l and chmod, I understood how read, write, and execute permissions control access. I also understood the difference between normal users and root, and why sudo must be used carefully.

3️⃣ One concept you are still confused about.

DNS resolution (Day 3) is slightly confusing to me. I understand that DNS converts domain names to IP addresses, but I would like to understand more deeply how the full DNS lookup process works step-by-step behind the scenes.

4️⃣ One mistake you made while completing tasks and what you learned from it.

On Day 1, while practicing Linux commands, I tried deleting a file without checking its name properly and got a “No such file or directory” error. It made me realize the importance of checking paths carefully using ls before performing actions. In cybersecurity and Linux, small mistakes can cause bigger problems.

5️⃣ What habit helped you stay consistent during this week?

Practicing daily without skipping tasks helped me stay consistent. Even when topics were slightly difficult (like networking and HTTP observation), I made sure to complete the task on the same day instead of postponing it. This built discipline.

6️⃣ What will you do differently in Week 2 to improve your learning?

In Week 2, I will:

Revise previous Linux and networking concepts before learning new topics.

Spend more time understanding practical tools.

Take short notes after every task.

Focus more on understanding real-world application instead of just completing the task.

7️⃣ What topic would you like to learn next in cybersecurity?

I would like to learn:

Network scanning tools like Nmap

Basic vulnerability scanning

How penetration testing works step-by-step

Basic web application security
